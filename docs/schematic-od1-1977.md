# Boss OD-1 Overdrive (1977) — Schematic 转录 / BOM / 网表

来源：electricdruid.net 重绘版 schematic（用户提供截图）。
注意：此重绘版运放画为 TL072；原厂 1977 OD-1 用的是四运放（RC3403A，后期改 µPC4741）。
仿真中运放宏模型参数做成可切换（TL072 / RC3403 / 4558），A/B 对比音色差异。

电源：VDD = 9V；VB（偏置）= 4.5V（VDD 分压）。

## 信号链

```
In ─ S1 输入缓冲(Q1 射随) ─ S2 Drive 增益+非对称削波(IC1.2) ─ S3 Filter 一阶低通(IC1.1)
   ─ S4 Level 电位器 ─ S5 输出缓冲(Q2 射随) ─ Out
```

（OD-1 无 Tone 级；SD-1 = OD-1 + Tone 级，后续升级点。）

## Drive 级连续时间方程（白盒推导）

以 VB 为参考零点（AC 分析），运放先按理想模型（v⁻=v⁺），后续替换宏模型：

状态变量：vC102（输入耦合电容）、vC104（接地支路电容）、vd = vo − v⁻（反馈网络两端电压）。

```
输入高通:   d(vC102)/dt = (vin − vC102) / (R104·C102),   v⁺ = vin − vC102
接地支路:   d(vC104)/dt = (v⁺ − vC104) / (R106·C104),    i_leg = (v⁺ − vC104)/R106
反馈 KCL:   C103·d(vd)/dt = i_leg − vd/Rf − iD(vd),      Rf = R105 + VR101
输出:       vo = v⁺ + vd
```

二极管（1N4148，SPICE 参数 Is=2.52nA, N=1.752）：
```
iD(vd) = Is·(exp( vd/(2·N·VT)) − 1)      … D101+D102 串联方向（有效理想因子 ×2）
       − Is·(exp(−vd/(  N·VT)) − 1)      … D103 反向
VT = kT/q ≈ 25.85mV @300K
```
正半周需 ~1.2V 才削、负半周 ~0.6V 即削 → 非对称削波，偶次谐波，OD-1/SD-1 音色标志。
