# Fender Princeton Reverb AA1164 — Schematic 转录 / BOM / 网表

来源：fender/AA1164_Princeton_Reverb_Annotated_Schematic.png（Rob Robinette 注释版原厂图）。
图注约定：电阻 1/2W ±10%（除非另注），电容 ≥400V（除非另注），电压 ±20%。

## 信号链

```
In ─ V1A(7025) 前级1 ─ 音色堆(T/B+固定中频) ─ Volume ─ V1B(7025) 前级2 ─┬─ 干路 3.3M∥10pF ─┐
                                                                        └ 500pF ─ V2(12AT7∥) 混响驱动 ─ TR3 ─ 弹簧箱 ─ V3A(½12AX7) 回收 ─ .003 ─ Reverb pot ─ 470k ─┤
                                                                                                                                                                   ├ V3B(½12AX7) 前级3 ←NFB(2700Ω→47Ω阴极尾)
                                                                                                                                                                   → V4B(½12AX7) cathodyne 倒相
                                                                                                                                                                   → 2×6V6GT 固定偏置推挽(−34V) ← V4A 颤音振荡器摇偏置
                                                                                                                                                                   → TR2(125A10B) 输出变压器 → 10" 喇叭
```

## 电子管清单

| 位号 | 管型 | 职能 |
|---|---|---|
| V1 | 7025（低噪 12AX7） | A: 前级1；B: 前级2 |
| V2 | 12AT7（两半并联） | 混响驱动（经 TR3 驱动弹簧箱） |
| V3 | 12AX7 | A: 混响回收；B: 前级3（NFB 汇入点） |
| V4 | 12AX7 | A: 颤音移相振荡器；B: cathodyne 倒相 |
| V5, V6 | 6V6GT ×2 | 推挽功放，固定偏置 −34V |
| V7 | 5U4GB | 全波整流 |

## 各级元件（图上直读）

### 输入 & V1A 前级1
- 输入：Hi/Low 插孔，68kΩ×2 栅阻，1MΩ 栅漏
- V1A：屏阻 100k（屏压 +160V，供电 D 节点 +240V），阴极 1500Ω ∥ 25µF/25V（+1.3V）

### 音色堆（FMV，无中频电位器）
- 高音电容 250pF；斜率电阻 100k；低音电容 .1µF；中频电容 .047µF；固定中频电阻 6800Ω
- Treble 250k-A、Bass 250k-A、Volume 1M-A（图注 pot 代码 2-35）

### V1B 前级2
- 屏阻 100k（+160V），阴极 1500 ∥ 25µF/25V；输出耦合 .02µF

### 混响支路
- 送出：500pF "Reverb Filter"（与 V2 1M 栅漏构成 ~318Hz 高通——只送中高频进弹簧，Fender 经典做法）
- V2 12AT7 两半并联，阴极 2200Ω ∥ 25µF/25V（+8.0V），屏供 +400V（B 节点）经 TR3（125A20B 混响变压器）
- 弹簧箱（reverb pan）+ 混响脚踏开关
- V3A 回收：屏阻 220k（+160V），阴极 1500 ∥ 25µF/25V（+1.2V）；输出 .003µF → Reverb 100k-L pot
- 混合节点（V3B 栅极）：干路 3.3M ∥ 10pF（bright），湿路 470k

### V3B 前级3 + NFB
- 屏阻 100k（+160V），阴极 1500 ∥ 25µF/25V + 47Ω NFB 尾（+1.2V）
- NFB：扬声器热端 → 2700Ω → 47Ω 顶端（阴极侧）
- 输出耦合 .02µF

### V4B cathodyne 倒相
- 栅漏 1M；屏阻 56k（+200V，C 节点 +320V 供电）；阴极 1k + 56k 尾（阴极 +50V）
- 双路输出：屏/阴极各经 .1µF → 220k 5% 栅漏 → 偏置线（−34V）→ 6V6 栅极

### 功放级
- 2× 6V6GT：屏 +410V（A 节点经 TR2 初级），帘栅 +400V（B 节点），固定偏置 −34V
- TR2 输出变压器 125A10B（推挽 6V6 典型 Ra-a ≈ 8kΩ : 8Ω）
- 偏置电源：PT 抽头 → 偏置整流二极管 → 100k 5% → 25µF/50V + 22k（−34V）

### 颤音（bias-wiggle）
- V4A 12AX7 移相振荡器：3 节 RC（.01/.01/.02 + 1M/220k），Speed 3M-RA，阴极 3300Ω ∥ 25µF
- 输出经 .02 → Intensity 250k-L → 注入偏置线（摇 6V6 偏置）；Vibrato 脚踏开关短路振荡器

### 电源
- PT（125P1B）340-0-340VAC → 5U4GB → A +420V（20µF/450V）
- A → 1k 1W → B +410/+400（帘栅、V2 屏，20µF）
- B → 18k 1W → C +320V（倒相、颤音，20µF）
- C → 18k 1W → D +240V（前级屏阻供电，20µF）
- 120Hz 全波纹波 + 整流管内阻 → 电源下垂（sag）为音色的一部分
